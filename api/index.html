    SupportKit Reference

    Overview

    The SupportKit SDK allows your app to provide delightful customer service and support experiences through

    • in-app conversations with users,
    • just-in-time answers,
    • and ubiquitous accessibility.

    There is also an optional integration with Zendesk to allow users to search through your knowledge base.

    The SupportKit class initializes the SDK, and shows the UI window.

    An SKTSettings object is passed to the initWithSettings: method to apply configuration options such as the app token, and the Zendesk knowledge base URL.

    The SKTUser class is used to attach custom user properties to each conversation.

    The SKTConversation class allows you to set a delegate for important changes, as well as read back certain information about the conversation.

    Read our installation guide to get started.
